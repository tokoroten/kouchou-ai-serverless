# レポートの公開(ホスティング)ガイド

広聴AI サーバレス版のレポートは「単一 HTML レポート」としてエクスポートすると、
Plotly・データ込みの**自己完結した 1 ファイル**になります(サーバ不要・オフラインでも開ける)。
このファイルを静的ホスティングに置くだけで、URL で共有できるレポートになります。

> **公開前の注意**: ホスティングに置いたレポートは**誰でも閲覧できます**。
> 元コメントの抜粋がレポートに含まれるため、個人情報や非公開情報が
> 含まれていないかを必ず確認してから公開してください。

## Netlify Drop で公開する(いちばん簡単)

[Netlify Drop](https://app.netlify.com/drop) はファイルをドラッグ&ドロップするだけで
静的サイトを公開できるサービスです。

1. レポート画面で **「単一 HTML レポート」** ボタンを押し、`○○.html` をダウンロードする。
2. ダウンロードしたファイルを `index.html` にリネームし、フォルダに入れる
   (Netlify Drop はフォルダ単位のアップロードで、トップに `index.html` を置くと
   サイトの URL 直下でレポートが開きます)。
3. ブラウザで <https://app.netlify.com/drop> を開く。
4. フォルダをページ中央のドロップエリアにドラッグ&ドロップする。
5. 数秒で `https://<ランダムな名前>.netlify.app` の公開 URL が発行される。

### 重要: アカウントに紐付ける

- **ログインせずにデプロイしたサイトは一定期間(24〜72時間程度)で自動削除されます。**
  継続して公開したい場合は、デプロイ後に表示される案内から Netlify アカウントを作成
  (無料)してサイトを自分のアカウントに紐付けてください。
- ログイン済みでデプロイすると最初から自分のサイトとして保存され、
  サイト名(URL のサブドメイン)も設定画面から変更できます。

### 更新するには

同じサイトを更新したい場合は、Netlify のサイト管理画面 → **Deploys** タブに
新しいフォルダをドラッグ&ドロップすると同じ URL のまま差し替えられます。

## その他の選択肢

| 方法 | 手軽さ | 備考 |
| --- | --- | --- |
| Netlify Drop | ◎ ドラッグ&ドロップ | 匿名デプロイは期限付き。アカウント紐付けで恒久化 |
| GitHub Pages | ○ リポジトリに push | 恒久 URL・履歴が残る。反映に約1分。git の知識が必要 |
| Cloudflare Pages | ○ ダッシュボードからアップロード | 無料枠が広い。アカウント必須 |
| GitHub Gist | × 不可 | raw が `text/plain` で返るため HTML として描画されない |

GitHub Pages の場合は、リポジトリ(例: `my-reports`)にエクスポートした HTML を
コミットし、Settings → Pages でブランチを指定すると
`https://<ユーザ名>.github.io/my-reports/<ファイル名>.html` で公開されます。
複数レポートを 1 リポジトリに追記していく運用に向いています。

## 補足

- 単一 HTML レポートは Plotly を同梱するため **数 MB** になりますが、
  各サービスの制限(Netlify: 1ファイル数百MB、GitHub Pages: 100MB/ファイル)には
  余裕で収まります。
- レポートはエクスポートした時点のスナップショットです。分析をやり直した場合は
  再エクスポートして差し替えてください。
- ブラウザの IndexedDB に保存されているレポート本体はエクスポートしない限り
  他の端末からは見えません。共有はこのページの方法で行ってください。
